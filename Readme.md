# CapW

CapW is a sample project that contains some core essentials for getting up and going quickly for new WinUI 3 projects based on the latest Windows App SDK. It aims to provide a clean foundation to build upon while not completely tying you into a specific architecture or framework.

## Draft

This project is in a draft state and subject to change. The intent to land upon a stable project template that has just enough abstraction to swap out implementations to keep up-to-date with the latest Windows App SDK releases.
